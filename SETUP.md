# Campus Swap Flutter App — Setup Instructions

## Prerequisites
- Flutter SDK installed (https://docs.flutter.dev/get-started/install)
- Android Studio with Flutter + Dart plugins
- Android emulator or physical device

## Steps to run in Android Studio

### 1. Open the project
Open Android Studio → **Open** → select the `campus_swap` folder.

### 2. Generate missing platform files
The `android/app/src/main/res/mipmap-*/` PNG launcher icons and a few other
platform files are generated by `flutter create` and cannot be stored as text.

Open a terminal inside Android Studio (**View → Tool Windows → Terminal**) and run:

```
flutter create --project-name campus_swap --org com.example .
```

This fills in any missing files without overwriting your existing `lib/` code.

### 3. Install dependencies
```
flutter pub get
```

### 4. Run
```
flutter run
```
Or use the **Run** button in Android Studio after selecting an emulator/device.

## Design
Matches the web app's brown/gold theme:
- **Espresso** `#4B3621` — primary text
- **Gold** `#D4AF37` — accent, active states
- **Cream** `#F7F2E7` — background
- Fonts: Manrope (headings) + Inter (body) via `google_fonts`
